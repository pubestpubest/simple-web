# Simple Web App

This project is a minimal static page styled with Pico CSS. It displays the current date and time under a headline labelled `App Version 1.0.0`.

## Getting Started

1. Clone or download this repository.
2. Install dependencies with `pnpm install`.
3. Start the local server:
   - Run `pnpm run start`.
   - Visit the URL shown in the terminal (default: `http://localhost:80`).

## Dependencies

- `http-server`: lightweight static file server used for local development.
- `Pico CSS`: lightweight css framework
